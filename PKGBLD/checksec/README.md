## checksec

Checksec is a bash script used to check for security features on system.

Options:

  --file ''executable-file''
  
  --dir ''directory'' [-v]
  
  --proc ''process name''
  
  --proc-all
  
  --proc-libs ''process ID''
  
  --kernel
  
  --fortify-file ''executable-file''
  
  --fortify-proc ''process ID''
  
  --version
  
  --help
